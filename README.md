Thomas Bonnin Dot Files
====================================

## DESCRIPTION

Configuraiton files (vim, zsh, git, etc...)


## INSTALLATION

    git clone git@github.com:TBonnin/dotfiles.git ~/.dotfiles
    cd ~/.dotfiles
    git submodule update --init
    rake install
